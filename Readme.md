# lego-webapp

> Next-gen frontend for abakus.no

## Setup
```bash
$ npm install
$ brew install forego
$ make watch
```

In order for `make watch` to work, you must have [foreman](https://github.com/ddollar/foreman), [forego](https://github.com/ddollar/forego) or similar installed.
